# 🚀 Career Coaching Sheffield – Conversion-Focused Website Redesign

## 📍 Project Summary

This project involved the strategic redesign and redevelopment of a career coaching website to address low engagement, weak conversion performance, and outdated user experience.

The objective was not just aesthetic improvement, but to deliver a **measurable increase in user engagement, trust, and client acquisition** through modern UX principles and conversion-focused design.

---

## ❗ Problem Statement

The original website functioned primarily as a static information page rather than a conversion tool. Key issues identified:

* High bounce rates due to lack of immediate value clarity
* Weak above-the-fold messaging (no defined value proposition)
* Limited trust signals despite strong real-world testimonials
* Poor user journey with no clear conversion funnel
* Outdated design reducing perceived credibility

---

## 📊 Key Hypotheses

The redesign was guided by the following assumptions:

1. Users decide whether to stay or leave within the first 5–10 seconds
2. Clear messaging and structure will significantly reduce bounce rate
3. Strong CTAs and guided flow will improve conversion rates
4. Social proof, when surfaced effectively, will increase trust and engagement

---

## 🧠 Approach & Methodology

### 1. User-Centric Analysis

* Identified core user segments:

  * Career changers
  * Job seekers struggling with applications
  * Professionals lacking clarity or direction

* Mapped key pain points:

  * Uncertainty about career direction
  * Low confidence in interviews
  * Repeated job rejections

---

### 2. Conversion-Focused Redesign

The website was rebuilt using a structured flow:

**Hero → Problem → Solution → Trust → Action**

Key improvements:

* Clear, outcome-driven headline
* Immediate call-to-action (Book Consultation)
* Simplified navigation and decision paths

---

### 3. UX/UI Modernisation

* Introduced clean, minimal layout with improved spacing
* Mobile-first responsive design
* Strong visual hierarchy for better scanability
* Reduced cognitive load by breaking large text blocks

---

### 4. Trust & Credibility Layer

* Integrated real Google testimonials into structured UI components
* Highlighted measurable client outcomes (career changes, job offers, confidence gains)
* Reinforced personal brand (approachability + expertise)

---

## 📈 Expected Impact & Success Metrics

While this project is a redesign case study, the improvements are aligned with industry benchmarks and expected performance gains:

### Engagement Metrics

* ⬇️ Bounce rate reduction: **30–50%**
* ⬆️ Average session duration: **+40%**
* ⬆️ Pages per session: **+25%**

### Conversion Metrics

* ⬆️ CTA click-through rate: **2–3x increase**
* ⬆️ Consultation bookings: **20–40% increase**

### User Behaviour Improvements

* Over **50% of users engaging within the first 60 seconds**
* Increased scroll depth due to structured storytelling
* Higher interaction with testimonial and service sections
